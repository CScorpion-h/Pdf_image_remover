# PDF 图片移除工具

一个用于分析并移除 PDF 中二维码、边角图片（Logo/页眉页脚）与重复图片（水印）的桌面工具。

##✨ 功能
- 交互模式：分析后预览候选图片，勾选需要删除的项并保存为新 PDF；
- 全自动模式：批量处理列表中的所有 PDF；
- 结束报告（自动模式）：明确告知每个文件是否有变更、移除数量与输出路径；
- 拖拽添加文件，支持多选。

##✨ 亮点
- 状态机式协调，实时进度；
- 更严格的小图过滤，减少无效任务；
- 兼容 Windows 7与 Windows 10/11。

##✨ 使用
1. 将 PDF 拖入列表或点击“添加文件”；
2. 勾选移除规则（二维码 / 边角图片 / 重复图片）；
3. 选择“分析并预览选中项”或“全自动处理所有文件”；
4. 交互模式：在预览窗口勾选要删除的图片后保存；
5. 自动模式：结束后弹出“批处理报告”，显示每个文件的处理结果与输出路径。
