# PDF 图片移除工具

这是一个简单易用的桌面工具，旨在帮助用户快速、智能地从 PDF 文件中移除不需要的图片，例如二维码、页眉/页脚的 Logo 以及重复出现的水印。

## ✨ 主要功能

- **支持移除多种图片**：
  - **二维码**：自动检测并移除 QR code。
  - **边角图片**：移除位于页面角落的图片，通常是 Logo、页眉或页脚装饰。
  - **重复图片**：移除在多页中重复出现的图片，通常是水印。
- **✅ 安全的预览与选择机制**：
  - 在执行任何删除操作前，会弹出一个预览窗口，展示所有待移除图片的缩略图。
  - 用户可以手动取消勾选任何不想删除的图片，完全避免误删。
- **⚡ 高效流畅的用户体验**：
  - **实时进度条**：分析过程分为“扫描”和“分析”两个阶段，进度条会实时反馈，让你清楚地知道程序正在工作。
- **⚙️ 个性化设置保存**：
  - 程序会自动保存你上次使用的移除规则，下次启动时无需重新勾选。
- **? 跨平台兼容**：
  - 提供为 Windows 10/11 和 Windows 7 优化的两个独立版本。

## ? 如何使用

1. 根据你的操作系统，下载对应的 `.exe` 文件 (`QRCodeRemover_win10.exe` 或 `QRCodeRemover_win7.exe`)。
2. 双击运行程序。
3. 点击 **“打开文件”** 并选择一个 PDF 文件。
4. 在 **“移除规则”** 区域，勾选你想要移除的图片类型。
5. 点击 **“分析并预览待移除图片”** 按钮，程序将开始在后台分析
6. 分析完成后，在弹出的 **预览窗口** 中，取消勾选任何你希望保留的图片。
7. 点击 **“确认移除并保存”**，然后选择一个位置来保存处理后的新 PDF 文件。
